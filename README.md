# BS-Archive v1.2 (internal build v7.4.8-buildfix)

Beat Saber / ScoreSaber 기록 분석과 랭크맵 기록 관리를 위한 한국어 기록 도구입니다.

## 이번 내부 패치(v7.4.5)

- 미기록 추천곡 목표 증가량 필터 허용 범위를 `±5pp`에서 `±1pp`로 더 엄격하게 변경
- 미기록 추천곡 명칭을 미기록 추천곡으로 변경
- 추천 탭의 후보 개수 표시 제거
- 목표 조건에 정확히 맞는 후보가 없으면 1pp 이상 증가 가능성이 있는 가장 가까운 후보를 표시
- 미기록 추천곡에서 다른 후보 보기 버튼 추가
- 미기록 추천곡 목표 증가량 기본값을 유저 총 PP에 따라 자동 설정
- 목표 증가량 슬라이더 최소값을 +1pp로 변경
- 초기화 버튼을 누르면 유저 실력대별 기본 목표값으로 복원
- 조건에 맞는 곡이 없으면 억지로 목록을 채우지 않고 안내 문구 표시
- `Top 30 성과표`를 이미지(PNG)로 저장하는 기능 추가
- 저장 이미지에 닉네임, 프로필 사진, 티어, 총 PP, 글로벌/국가 랭킹, 평균 정확도, Top PP 포함
- 추천곡 탭의 `.bplist` 저장 기능 유지

## 주요 기능

- ScoreSaber ID / 커스텀 ID / 프로필 URL 기반 유저 기록 분석
- 총 PP, 글로벌 랭킹, 국가 랭킹, 평균 정확도, Top PP 표시
- BS-Archive 티어 표시
- Top 30 카드형 성과표
- 기록 갱신 추천 / 미기록 추천곡 / 오래된 기록 추천
- 추천곡 `.bplist` 플레이리스트 저장
- Top 30 성과표 PNG 저장
- Top 30 PNG 저장 레이아웃을 6열 압축형으로 변경해 이미지 길이를 줄임
- BS-Archive 기록 저장소(수동 기록, 메모, JSON 백업/복원)

## 실행

새 폴더에서 처음 실행할 때:

```bash
npm install
npm run dev
```

기존 폴더에서 파일만 덮어썼다면:

```bash
npm run dev
```

## 주소

- 분석기: http://localhost:3000
- 기록장: http://localhost:3000/logbook

## 내부 패치 v7.4.6
