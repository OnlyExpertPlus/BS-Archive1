# BS-Archive v1.1

Beat Saber / ScoreSaber 기록 분석과 랭크맵 기록 관리를 위한 한국어 기록 도구입니다.

ScoreSaber ID, 커스텀 ID, 프로필 URL을 입력해 유저 기록을 분석하고, 추천곡과 성과표를 확인할 수 있습니다.  
또한 BS-Archive 기록장에서 랭크맵별 기록, L/R 평균 컷, 메모, FC 여부를 따로 저장할 수 있습니다.

## 주요 기능

- ScoreSaber ID / 커스텀 ID / 프로필 URL로 유저 기록 분석
- 총 PP, 글로벌 랭킹, 국가 랭킹, 평균 정확도, Top PP 표시
- BS-Archive 전용 PP 티어 표시
- Top 30 카드형 성과표 제공
- 난이도 구간별 기록 통계 제공
- ScoreSaber 랭크맵 기반 추천곡 제공
  - 기록 갱신 추천 TOP 10
  - 미기록 PP 효율곡 TOP 10
  - 오래된 기록 갱신 후보 TOP 10
- ScoreSaber PP 커브 기반 추정 PP와 예상 증가량 표시
- BeatSaver / ScoreSaber 바로가기 버튼 제공
- 랭크맵별 기록 저장
- L/R 평균 컷, 메모, FC 여부 수동 입력
- 이전 기록 대비 ACC, 점수, PP, L/R 평균 컷 증감량 표시
- 유저별 기록 분리 저장
- JSON 백업 / 복원 지원

## v1.1 업데이트 내용

- 분석 페이지 이름을 `BS-Analyzer`로 정리했습니다.
- 기록 저장소 페이지 이름을 `BS-Archive`로 정리했습니다.
- 추천곡 영역을 탭 방식으로 정리했습니다.
- 추천곡 카테고리를 3개로 구성했습니다.
  - 기록 갱신 추천
  - 미기록 PP 효율곡
  - 오래된 기록 갱신 후보
- ScoreSaber PP 커브와 별당 상수를 이용해 추정 PP 계산을 개선했습니다.
- 추천곡에 예상 PP 증가량을 표시합니다.
- 예상 PP와 증가량은 실제 ScoreSaber 반영값과 다를 수 있습니다.
- 새로운 BS-Archive 티어표를 적용했습니다.
  - IRON III ~ LEGEND
- CHAMPION 이상 티어에 그라데이션 효과를 적용했습니다.
- 프로필 통계 카드를 5개로 정리했습니다.
  - 총 PP / 글로벌 랭킹 / 국가 랭킹 / 평균 정확도 / Top PP
- 오래된 기록 갱신 후보는 90일 이상 지난 기록만 표시하도록 수정했습니다.
- ScoreSaber 실제 랭크맵이 없는 별 구간은 자동으로 숨기도록 수정했습니다.
- 기록은 플레이어 ID별로 분리 저장됩니다.
- L/R 자동 보완 기능은 안정화 전까지 제거하고, 수동 입력 방식으로 유지했습니다.
- localStorage 저장 용량 문제를 줄이기 위해 분석용 대용량 데이터는 저장하지 않도록 수정했습니다.

## 실행 방법

처음 실행하거나 새 폴더에 압축을 푼 경우, 한 번 설치가 필요합니다.

```bash
npm install
npm run dev

이미 node_modules가 있는 기존 폴더에 파일만 덮어쓴 경우에는 아래만 실행해도 됩니다.

npm run dev
로컬 주소
BS-Analyzer: http://localhost:3000
BS-Archive: http://localhost:3000/logbook
저장 방식 안내

현재 기록은 로그인 없이 브라우저의 localStorage에 저장됩니다.

같은 브라우저에서는 기록이 유지됩니다.
브라우저 데이터를 삭제하면 기록이 사라질 수 있습니다.
다른 기기와 자동 동기화되지는 않습니다.
중요한 기록은 JSON 백업 기능을 사용해 저장해두는 것을 권장합니다.
주의 사항

추천곡의 추정 PP와 예상 증가량은 ScoreSaber PP 커브 기반 계산값입니다.
실제 ScoreSaber 반영 PP, 기록 순위, 내부 반올림 방식에 따라 실제 결과와 차이가 있을 수 있습니다.
