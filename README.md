# BS-Archive v1.3 (internal build v8.1.3)

Beat Saber / ScoreSaber 기록 분석과 랭크맵 기록 관리를 위한 한국어 기록 도구입니다.

패치 내역
## 내부 빌드 v8.0

- 미기록 추천곡에서 이미 기록이 있는 곡이 섞이는 문제를 줄이기 위해 Top 기록 여러 페이지와 최근 기록까지 함께 비교합니다.
- 기록 갱신 추천을 전면 개편해 상위 고점 기록만이 아니라 실력 대비 낮은 기록도 우선 추천하도록 수정했습니다.
- 기록 갱신 추천과 오래된 기록 추천에 다른 후보 보기 기능을 추가했습니다.
- 오래된 기록 기준을 1주 / 1달 / 3달 / 6달 / 1년 중 선택할 수 있게 했습니다.
- BS-Archive 기록장에 별 구간별 요약표를 추가했습니다.
- Dashboard에 오늘의 추천 3곡을 추가했습니다.


## 내부 빌드 v8.0.1

- 오래된 기록 버튼을 `n일 이상` 누적 기준이 아니라 구간 기준으로 수정했습니다.
- 1주~1달 / 1달~3달 / 3달~6달 / 6달~1년 / 1년 이상으로 나눠 표시합니다.

## 내부 빌드 v8.0.2

- BS-Archive 별 구간별 요약표 집계 방식을 수정했습니다.
- leaderboardId 기준으로 맵을 중복 제거하고, 같은 맵의 여러 기록은 최고 ACC/PP 기록 하나만 집계합니다.
- 카드 목록과 요약표가 같은 기준으로 계산되도록 맞췄습니다.


## 내부 빌드 v8.0.3

- ScoreSaber 기록 ACC 계산을 modifier 보너스가 포함된 modifiedScore 기준이 아니라 baseScore 기준으로 수정했습니다.
- modifier 때문에 100%를 초과하던 기록은 표시/집계 시 100% 이하로 보정합니다.
- 별 구간 요약표의 Over 98% / Over 95% / Over SS / Over S 집계가 순수 ACC 기준으로 계산되도록 수정했습니다.


## 내부 빌드 v8.1

- ScoreSaber 기록 가져오기 시 같은 scoreId의 기록이 변경된 경우 이전 기록을 히스토리 스냅샷으로 보존합니다.
- 다시 기록 가져오기를 했을 때 이전 기록과 새 기록이 함께 남도록 히스토리 보존 로직을 강화했습니다.
- 오늘의 추천 3곡에 다시 뽑기 버튼과 슬롯머신 느낌의 리롤 애니메이션을 추가했습니다.
- 추천곡 제목에서 TOP 10 문구를 제거했습니다.
- modifier 기록 ACC 계산을 baseScore 정상 범위 확인 후 modifiedScore / multiplier 역산까지 사용하도록 보강했습니다.


## 내부 빌드 v8.1.1

- 오늘의 추천 3곡 알고리즘을 조정했습니다.
- 워밍업 추천은 유저 총 PP 구간에 맞춘 별 범위에서만 고릅니다.
  - 13000pp대는 8~9.5★
  - 14000pp대는 9~10.5★
  - 15000pp 이상은 최대 11★까지
- PP 도전 추천은 너무 낮은 별에서 97~98%를 요구하는 후보를 줄이고, 유저 실력대에 맞는 도전 별 범위를 우선합니다.


## 내부 빌드 v8.1.2

- 오늘의 추천 3곡 중 PP 도전 추천을 조정했습니다.
- 13000pp대 유저에게 9★대 고정확도 후보가 과하게 나오던 문제를 줄였습니다.
- PP 도전은 더 높은 별 구간에서 낮은 목표 정확도로 도전할 수 있는 곡을 우선 추천합니다.
- 13000pp대는 10.2~12.5★, 14000pp대는 10.2~13★, 15000pp 이상은 10.5★ 이상 도전 구간을 우선합니다.


## 내부 빌드 v8.1.3

- ScoreSaber 랭크 기록 ACC를 PP와 별 값으로 역산해 modifier 영향을 제거하도록 변경했습니다.
- baseScore/modifiedScore는 PP 역산이 불가능할 때만 보조 계산으로 사용합니다.
- 가져온 점수 값도 modifiedScore 대신 순수 점수 값을 저장합니다.
- 이전 버전에서 같은 ScoreSaber 기록이 100%로 잘못 저장된 경우 다시 가져오면서 교정하고, 잘못된 100% 히스토리도 제거합니다.

## 주요 기능

- ScoreSaber ID / 커스텀 ID / 프로필 URL 기반 유저 기록 분석
- 총 PP, 글로벌 랭킹, 국가 랭킹, 평균 정확도, Top PP 표시
- BS-Archive 티어 표시
- Top 30 카드형 성과표
- 기록 갱신 추천 / 미기록 추천곡 / 오래된 기록 추천
- 추천곡 `.bplist` 플레이리스트 저장
- Top 30 성과표 PNG 저장
- Top 30 PNG 저장 레이아웃을 6열 압축형으로 변경해 이미지 길이를 줄임
- BS-Archive 기록 저장소(수동 기록, 메모, JSON 백업/복원)

## 실행

새 폴더에서 처음 실행할 때:

```bash
npm install
npm run dev
```

기존 폴더에서 파일만 덮어썼다면:

```bash
npm run dev
```

## 주소

- 분석기: http://localhost:3000
- 기록장: http://localhost:3000/logbook
